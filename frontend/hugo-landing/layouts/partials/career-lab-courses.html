PROJECTS

5 Hands-On Projects to Build a Professional Portfolio

Project 1: Job Aid

Design a one-pager job aid focused on quick reference support for a specific work scenario.

Project 2: Training Proposal

Conduct needs analysis and create a formal proposal, including problem statement, learning objectives, and recommended solution.

Project 3: Professional Deliverables

Create a detailed learning solution blueprint, including learning objectives, assessment strategies, and accessibility considerations.

Project 4: Design Document

Build a responsive, single-learning-objective online learning experience using Articulate Rise.

Project 5: Interactive Scenario

Develop immersive branching scenarios using Storyline/H5P, where learners make decisions and see the consequences.

Portfolio 6: Portfolio Website

Design a professional website showcasing all 5 projects, highlighting the design process and skills through case studies.
